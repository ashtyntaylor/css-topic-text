Text CSS Properties

Text-Align

The text-align property specifies the horizontal alignment of text in an element.

See the Pen Text-Align by e-b11 (@e-b11) on CodePen.

Text-Decoration

The text-decoration property sets the appearance of decorative lines on text. It is shorthand for text-decoration-line, text-decoration-color, text-decoration-style, and the newer text-decoration-thickness property.

Text-Transform

The text-transform property specifies how to capitalize an element's text. It can be used to make text appear in all-uppercase or all-lowercase, or with each word capitalized.

See the Pen text-indent-keyword by ashtyntaylor (@ashtyntaylor) on CodePen.

Text-Indent

The text-indent property sets the length of empty space (indentation) that is put before lines of text in a block.

text-indent is specified by 2 different values

  1. length values
  2. percentage values

1. Length

Length represents a distance value. It consists of a number followed by a unit of measurement.

See the Pen Untitled by ashtyntaylor (@ashtyntaylor) on CodePen.

2. Percentage

Represents a percentage value. It is often used to define a size as relative to an element's parent object

See the Pen Untitled by ashtyntaylor (@ashtyntaylor) on CodePen.

Line-Height

The line-height property sets the height of a line box. It is commonly used to set the distance between lines of text.

line-height can be specified in four ways:

  1. number
  2. length
  3. percentage
  4. the keyword normal

1. Number:

See the Pen Number by Ava (@Avareesew) on CodePen.

Notes: numbers with no units calculates the line height from the font-size. (ex: 30 px * 1.1=33 px). It is better to use unitless line-heights because using units can create issues because of inheritance issues.

2. Length:

See the Pen Length by Ava (@Avareesew) on CodePen.

3. Percentage:

4. Keyword Normal:

See the Pen Normal by Ava (@Avareesew) on CodePen.

**The general rule is to use a minimum line-height of 1.5(number) for accessibility reasons.**
